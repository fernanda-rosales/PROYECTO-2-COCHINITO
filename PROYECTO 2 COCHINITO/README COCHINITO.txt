Para correr el programa Alcancia Digital se debe de:

instalar el IDE netbeans 8.2 , pero antes debe tener instalado en el 
sistema la actualización 19 de Java SE Development Kit (JDK) 

Si no tiene una instalación de JDK, no puede continuar con la instalación del IDE,
posteriormente ejecutar el archivo exe y tendra intalado el id donde fue creada
nuestra aplicacion y donde podra ser ejecutada 

Una ves instalando el entorno de desarrollo netbeans buscar en la carpeta el archivo
con el nombre CochinitoDigital que se anexa en esa carpeta se encuantra nuestro
proyecto con las librerias necesarias para correr.

abrir el entorno de netbeans y dar click en file, despues open project y buscar la carpeta donde
se encuentra el archivo CochinitoDigital una vez abierto dar click en run project y se
compilara el proyecto teniendo su ejecucion, el programa maneja archivos por lo cual
cuando se abra la alcancia tendra un registro de propietario y fecha si desesea un nuevo
archivo para ingresar nuevos datos debe darse click en romper alcancia y se 
creara un nuevo archivo.txt(en la carpeta anexda se debe encontrar dos archivos de texto
llamados Cochinito y Fecha) ya que son los nombres en donde se guarda la informacion

ademas del programa se anexa un instalador con nombre setup solo se tiene que darle doble click
y se podra instalar automaticamente en la computadora sin descargar netbeans.
